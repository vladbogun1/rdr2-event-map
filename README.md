# RDR2 Event Map (GitHub Pages)

Это статический сайт на Leaflet, который показывает вашу кастомную карту RDR2 (как тайлы) + метки из `markers.json`.

## Как включить GitHub Pages
1. Залей весь проект в репозиторий (например `rdr2-event-map`)
2. GitHub → **Settings** → **Pages**
3. Source: **Deploy from a branch**
4. Branch: **main** / folder: **/(root)**
5. Ссылка будет вида: `https://<username>.github.io/<repo>/`

## Где править метки
- Открой `admin.html`
- Клик по карте — добавить
- Перетащи маркер — переместить
- Потом нажми **Export markers.json** и замени файл `markers.json` в репозитории (commit/push)

> Важно: GitHub Pages — статический хостинг, он **не может** записывать изменения обратно на сервер.
> Поэтому “сохранение” делается через экспорт файла.

## Замена карты
- Если меняешь исходную карту, нужно заново нарезать тайлы.
- В этом проекте тайлы уже сгенерированы для размера **2048×1593**, `maxZoom=3`.

## Структура
- `index.html` — просмотр
- `admin.html` — редактирование (экспорт/импорт JSON)
- `markers.json` — метки
- `tiles/` — тайлы карты (`z/x/y.webp`)
